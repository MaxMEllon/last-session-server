# Server side for [last session](https://github.com/MaxMEllon/last-session)

Quick start in local
---

1. npm i -g saku
1. saku init
1. saku dev

Tasks
---

Configured tasks by [saku](https://github.com/kt3k/saku).

- **dev** : `saku dev`
  > start develop server

Global dependencies
---

- **node** >= v8.5.0
- **npm** >= v5.5.1
- **saku** >= v0.11.0
